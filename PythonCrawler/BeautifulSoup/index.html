Beautiful Soup

"A tremendous boon." -- Python411 Podcast

Beautiful Soup is a python library designed for quick turnaround peojects like screen-scraping. There features make it powerful:

  1. Beautiful Soup provides a few simple methods and Pythonic idioms for navigating, searching, and modifying a parse tree: a tookit for dissecting a document and extracting what you need. It doesn't take much code to write an application.
  2. Beautiful Soup automatically converts incoming documents to unicode and outgoing documents to UTF-8. You don't have to think about encoding, unless the document doesn't specify an encoding and Beautiful Soup can't detect one. Then you just have to specify the original encoding.
  3. Beautiful Soup sits on top of popular Python parsers like Lxml and html5lib, allowing you to try out different parsing strategies or trade speed for flexibility.

Beautiful Soup parses anything you give it, and does the tree travelsal stuff for you. You can tell it "Find all the likns", or "Find all the links of class externaLink", or "Find all the links whose urls match "foo.com", or "Find all the table heading that's got bold text, then given me that text."

Valueable data that was once locked up in poorly-designed websites is now within you reach. Projects that would have taken hours yake only munites withn Beautiful Soup
